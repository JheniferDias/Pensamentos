## Introdução

Esse app foi desenvolvido no intuito de apresentar ao usuário vários pensamentos de grandes filósofos, e pessoas importantes na histó
ria da humanidade, para realizar reflexões e até
 mesmo leva-lo ao um autoconhecimento. 
 
 ## Aprendizados
 
 - Uso de Settings Bundle
 - Uso do UISwitch
 - Uso do UISlider
 - Uso do UISegmentedControl
 - Salvando dados com uso de UserDefaults


## Requerimentos

- Xcode 13.1
- Swift 5
